# Todo API 🚀

A REST API built with Node.js and Express with full CRUD operations.

## Endpoints

| Method | Route | Description |
|--------|-------|-------------|
| GET | /tasks | Get all tasks |
| POST | /tasks | Create a new task |
| PUT | /tasks/:id | Update a task |
| DELETE | /tasks/:id | Delete a task |

## Technologies
- Node.js
- Express.js

## Concepts Applied
- REST API architecture
- CRUD Operations
- Route parameters
- Middleware (express.json)
- In-memory data storage

## How to run
```bash
npm install
node index.js
```
